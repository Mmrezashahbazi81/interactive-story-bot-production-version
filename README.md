# Interactive Story Bot

A production-ready Interactive Story Bot built with Python, FastAPI and Docker.

## Features

- Async FastAPI backend
- Bale Bot API
- Dockerized deployment
- SQLite persistence
- Nginx Reverse Proxy
- HTTPS (Let's Encrypt)
- Webhook based updates
- Story JSON Loader
- Multiplayer Lobby System
- Ready System
- Membership Verification
- Optimized SQLite
- Production Logging

---

## Tech Stack

- Python 3.11
- FastAPI
- httpx
- aiosqlite
- Docker
- Docker Compose
- Nginx
- Certbot
